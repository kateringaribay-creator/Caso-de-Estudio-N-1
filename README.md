# Caso-de-Estudio-N-1
# 📊 Bank Marketing - Análisis Exploratorio de Datos (EDA) con Streamlit

## 📌 Descripción del proyecto

Este proyecto consiste en una aplicación interactiva desarrollada en **Python con Streamlit**, orientada al **Análisis Exploratorio de Datos (EDA)** del dataset *BankMarketing.csv*, correspondiente a una campaña de marketing de una entidad financiera.

El objetivo principal es analizar los factores que influyen en la aceptación de la campaña de marketing (variable objetivo `y`), mediante técnicas de análisis estadístico y visualización de datos.

Este proyecto no incluye modelos predictivos, sino que se enfoca en la exploración, comprensión y visualización de los datos para generar insights relevantes.
